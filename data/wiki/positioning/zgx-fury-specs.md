# HP ZGX Fury - Product Specs

**Last Updated**: 2026-05-14
**Product Tier**: Fury

---

## Overview

The HP ZGX Fury is a liquid-cooled, on-premises AI workstation built on
the NVIDIA Blackwell Ultra platform. It delivers data-center-class AI
inference entirely on-premises with zero cloud dependencies.

## Specs

| Spec | Value |
|------|-------|
| cpu | NVIDIA Grace (ARM64 Neoverse, NOT x86) |
| gpu | NVIDIA Blackwell Ultra |
| memory | 748GB coherent (unified CPU+GPU) |
| cooling | Liquid-cooled |
| price_range | ~$100K |
| architecture | ARM64/aarch64 |

## Key Differentiators

- ZGX Toolkit and HP Z Runtime offer tools to get you started with development and inference that are free in opposition to all   other OEMs that offer NVIDIA AI Enterprise which is $4500/year/unit

## What are not key differentiators

- The hardware, all OEMs have the same hardware
- Compliance by Architecture: data never leaves the device; all OEMs can promise that
